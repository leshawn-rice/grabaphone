Endpoints

GET /api/get-manufacturers

    Requirements:
  • API Key
    Optional arguments:
  • Name (get a specific manufacturer by name)
  • Limit (limit results to X manufacturers, with a max limit of 100)

    Example Response

GET /api/get-phones

    Requirements:
  • API Key
    Optional arguments:
  • Name (get a specific phone by name (may result in more than one phone if sharing name))
  • Limit (limit results to X phones, with a max limit of 100)
  • Price (get phones based on a price range)

    Example Response
