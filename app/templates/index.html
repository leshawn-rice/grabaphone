Endpoints

GET /api/get-manufacturers

Get up to 100 manufacturers & all their devices. Requires an API Key, and accepts a manufacturer name and a limit (up to 100) to narrow the results

    Requirements:
  • API Key

    Optional arguments:
  • manufacturer (get a specific manufacturer by name)
  • limit (limit results to X manufacturers, with a max limit of 100)

GET /api/get-devices

Get up to 100 devices. Requires an API Key, and accepts a manufacturer name, a part of a device name (iPhone, Galaxy A, Moto e etc.), and a limit (up to 100) to narrow the results

    Requirements:
  • API Key

    Optional arguments:
  • name (Get a list of devices that include "name" in their name )
  • limit (Limit results to "limit" devices, with a max limit of 100)
  • manufacturer (Limit results to devices made by "manufacturer")

GET /api/get-latest-devices

Get up to 100 of the latest devices. Requires an API Key, and accepts a manufacturer name, a part of a device name (iPhone, Galaxy A, Moto e etc.), and whether or not the device has been released to narrow the results

    Requirements:
  • API Key

    Optional arguments:
  • name (Get a list of devices that include "name" in their name )
  • manufacturer (Limit results to devices made by "manufacturer")
  • is_released (Limit results to devices already released. Leave blank if you want unreleased devices)
