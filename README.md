# Grabaphone!

A RESTful API that accepts and responds with JSON    [![Build Status](https://travis-ci.com/leshawn-rice/grabaphone.svg?branch=main)](https://travis-ci.com/leshawn-rice/grabaphone) [![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT) [![GitHub issues open](https://img.shields.io/github/issues/leshawn-rice/grabaphone)](https://github.com/leshawn-rice/grabaphone/issues)

## Grabaphone will allow developers to get cellular device data from 5000+ devices!
## Device data includes
 - Device Manufacturer
 - Device Name
 - Device Rating 
 - Device Specifications
 - Device Image (Link)

## Going live by the end of February 2021!
